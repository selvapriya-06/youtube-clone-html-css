# YouTube Clone (HTML & CSS)

A responsive YouTube homepage clone built using HTML and CSS.

## Features
- Responsive layout
- Sidebar navigation
- Video thumbnails
- Header with search bar
- Channel profile section

## Technologies Used
- HTML5
- CSS3

## Project Structure
├── channel-picture/
├── icons/
├── styles/
├── thumbnails/
└── youtube.html

This project was built while learning HTML and CSS and helped me understand:
- Flexbox
- CSS Grid
- Positioning
- Responsive layouts
- Web page structuring
